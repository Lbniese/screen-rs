# Provenance

No upstream GNU Screen source files, tests, or documentation have been copied or
adapted into this repository.

Reference behavior must be obtained by running a configured GNU Screen
executable during differential tests.

The compatibility feature manifest `compatibility/features/screen-5.0.2.toml`
contains an inventory derived from GNU Screen command-line help output and the
screen-rs parser/runtime command surface. No upstream GNU Screen source files or
tests were copied to create it.
